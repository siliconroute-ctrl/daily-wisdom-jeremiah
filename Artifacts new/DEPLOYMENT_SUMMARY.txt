📞 QUICK TROUBLESHOOTING
================================================================================

npm install fails?
  → rm -rf node_modules && npm install

Can't run locally?
  → Make sure Node 16+ installed: node --version
  → Make sure you're in project folder: cd daily-wisdom-jeremiah

Deployment to Vercel fails?
  → Make sure code is pushed: git push
  → Check GitHub repo is public
  → Check Vercel can see your repo

Can't sign up?
  → Check browser console (F12 → Console)
  → Verify Firebase authentication enabled
  → Check internet connection

No verses showing?
  → Create "verses" collection in Firebase Firestore
  → Add documents from jeremiah-verses-db.json
